LC Chatbot

A beautiful, embeddable chat widget for any website

💬 Markdown Support

Renders rich markdown responses including code blocks, lists, and links.

📐 Resizable

Drag to resize the chat panel. Size is remembered across sessions.

📜 History

Infinite scroll to load older messages with date markers.

🎨 Themeable

Customize colors and styles with CSS custom properties.

Demo Controls

Note: This demo uses a mock API endpoint. To connect to a real backend, update the api-base-url attribute. The widget expects:

• POST /chat - Send message, receive markdown response
• GET /history - Load conversation history
